# Interface Lab

Стартовая версия персонального портфолио для веб-дизайнера и веб-разработчика.

## Что уже есть

- Главный экран с сильной презентацией.
- Блок избранных работ с фильтрами.
- Секция для фото, видео, ссылок на Figma, GitHub, PDF и другие материалы.
- Описание процесса работы.
- Контакты для работодателей и клиентов.
- Адаптивная версия для мобильных устройств.

## Как запустить

Открой `index.html` в браузере или запусти локальный сервер из папки проекта.

```powershell
python -m http.server 8000
```

После запуска сайт будет доступен по адресу `http://localhost:8000`.

## Что заменить под себя

- В `index.html` замени имя, описание, контакты и ссылки.
- В секции `#works` добавь реальные проекты, скриншоты, роли и результаты.
- В секции `#media` добавь видео, фото, презентации или ссылки на внешние материалы.
- В `styles.css` можно менять цвета, шрифты и визуальный стиль.

## Куда складывать материалы

- `assets/works/` — скриншоты проектов, обложки кейсов и мокапы.
- `assets/media/` — видео, фото, записи экрана и демонстрации.
- `assets/docs/` — PDF, презентации, резюме и дополнительные файлы.

## Как добавить проект на сайт

Самый простой способ — запустить мастер добавления проекта:

```powershell
npm.cmd run add-project
```

Он спросит название, категорию, описание, ссылки и путь к картинкам. Можно указать одну папку с изображениями или несколько файлов через запятую. Мастер сам скопирует изображения в `assets/projects/`, обновит `data/projects.js` и добавит проект первым в списке.

После проверки сайта останется опубликовать изменения:

```powershell
git add -A
git commit -m "Add new portfolio project"
git push
```

GitHub Pages обновит сайт автоматически.

### Ссылки с иконками

В мастере добавления или редактирования ссылки вводятся через запятую:

```text
GitHub=https://github.com/user/repo, Figma=https://figma.com/file/..., Сайт=https://example.com
```

Иконка подставится автоматически по названию или адресу. Поддерживаются `GitHub`, `Figma`, `Behance`, `Dribbble`, `Demo`, `Сайт`.

Если нужно указать иконку вручную, добавь её через `|`:

```text
Документация|link=https://example.com/docs, Прототип|figma=https://figma.com/file/...
```

## Как редактировать проект

Запусти мастер редактирования:

```powershell
npm.cmd run edit-project
```

Он покажет список проектов и попросит номер карточки. Если поле менять не нужно, просто нажми `Enter`. Можно обновить название, категорию, год, роль, описания, ссылки и изображения.

После проверки сайта опубликуй изменения:

```powershell
git add -A
git commit -m "Update portfolio project"
git push
```

## Как удалить проект

Запусти мастер удаления:

```powershell
npm.cmd run remove-project
```

Он покажет список проектов, попросит номер карточки и подтверждение. После удаления можно отдельно подтвердить удаление папки с изображениями проекта.

После проверки сайта опубликуй изменения:

```powershell
git add -A
git commit -m "Remove portfolio project"
git push
```
